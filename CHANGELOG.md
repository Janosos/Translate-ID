# Changelog

## [1.0.0] - 2024-05-22
### Added
- Initial release of the MX-US Translator plugin.
- Custom floating toggle with embedded SVG flags (Mexico & USA).
- Google Translate API integration restricted to English and Spanish.
- CSS styles for hiding the default Google Translate banner.
- Cookie-based language persistence logic.
